<h1 align="center">💰 Expense Tracker App</h1>

<p align="center">
A modern <b>Expense Tracker Android App</b> built using <b>Jetpack Compose</b> to manage daily expenses and track spending habits.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>➕ Add income & expenses</li>
  <li>📂 Category-based transactions</li>
  <li>📊 Expense visualization with charts</li>
  <li>📅 Track transactions by date</li>
  <li>💾 Offline storage using Room Database</li>
  <li>⚡ Smooth UI with Jetpack Compose</li>
  <li>🔄 Real-time updates</li>
</ul>

<hr>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li><b>Language:</b> Kotlin</li>
  <li><b>UI:</b> Jetpack Compose</li>
  <li><b>Architecture:</b> MVVM</li>
  <li><b>Database:</b> Room</li>
  <li><b>Dependency Injection:</b> Hilt</li>
  <li><b>Networking:</b> Retrofit</li>
</ul>
